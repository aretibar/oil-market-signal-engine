# Oil Market Signal Engine

This project analyzes real-world energy data (EIA) to generate insights and signals about the oil market.

## 📊 Data Sources
- Inventory data
- Production data
- Oil prices

## ⚙️ Features
- Data cleaning and merging
- Time-series analysis
- Signal generation based on supply/demand trends

## 🛠️ Tech Stack
- Python
- Pandas
- Matplotlib

## 🚀 How to Run

```bash
pip install -r requirements.txt
python src/app.py
